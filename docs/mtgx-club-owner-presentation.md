# MTGx — Презентация для владельца клуба

---

## Что это

MTGx — платформа для координации MTG-игр. Помогает игрокам находить игры, а клубам — заполнять столы. Работает как приложение на телефоне (PWA), вход через Google в один тап.

Адрес: **mtgx.app**

---

## Что уже работает

### Для игроков

**Регистрация за минуту.** Ссылка в WhatsApp → вход через Google → город, форматы, расписание → лента событий. Никаких паролей и форм.

**Профиль.** Имя, аватар (загрузка своего фото или автоматически из Google), город, форматы (Pauper / Commander / Standard / Draft), расписание доступности (утро/день/вечер по дням недели), WhatsApp для связи, наличие машины, интерес к обмену картами.

**Лента событий.** Все ближайшие игры с фильтрами по формату и городу. На карточке: формат, время, место, сколько записалось, сколько мест осталось. Круговая диаграмма заполненности.

**Онлайн и гибридные события.** Три режима: вживую, онлайн, гибрид. Поддержка платформ: SpellTable, MTGO, MTG Arena, Discord, Zoom. Ссылка для подключения прямо в карточке события.

**Запись на события (RSVP).** Три варианта: точно иду / может быть / не приду. Счётчик обновляется мгновенно у всех. При максимуме — автоматическая очередь ожидания (waitlist) с позицией в очереди. Место освободилось — первый в очереди получает push и автоматически переносится в участники.

**«Иду сегодня».** Кнопка "я свободен прямо сейчас" — выбираешь формат и длительность (1-5 часов). Сигнал виден всем в городе. Когда 3+ человек нажали — подсказка "Создать встречу?". Заменяет 45-минутный WhatsApp-тред "кто сегодня?".

**Теги настроения.** На каждом событии видно: казуальная / соревновательная / тест колоды / обучение. Игрок понимает куда идёт.

**Политика прокси.** На событии указано: без прокси / частично / полностью разрешены. Убирает типовые вопросы.

**Commander Brackets.** При записи на Commander-событие игрок указывает силу колоды (1-5). Организатор видит распределение за столом до начала игры. Решает главную проблему Commander — мисматч по силе.

**Подписки.** Три типа: на организатора, на клуб, на формат+город. Одно уведомление на событие, без дубликатов.

**Автоматический подбор игр.** Настраиваешь один раз: какие форматы, какие типы событий (турниры/встречи), расписание по дням (утро/вечер), радиус поиска (мой город/рядом/все), макс. уведомлений в день. Получаешь push только когда появляется подходящая игра.

**Система приглашений.** Организатор видит рекомендуемых игроков (подходят по формату, городу, доступности) и приглашает одной кнопкой. Игроки могут звать друг друга напрямую. Гибкие настройки приватности: принимать от всех / только от тех, с кем играл / только из своих клубов / никого. Режим "Не беспокоить" с таймером.

**QR Check-in.** Организатор показывает QR на телефоне, игроки сканируют — подтверждение реальной явки. На основе этого считается рейтинг надёжности.

**Рейтинг надёжности.** Автоматический расчёт: сколько раз записался и реально пришёл. Виден только организаторам. Без штрафов — просто информация для планирования.

**WhatsApp-шеринг.** Кнопка на любом событии — генерирует сообщение с названием, форматом, датой, местом и ссылкой. Один тап — в чат.

**Обратная связь.** Кнопка "Сообщить о проблеме" с любого экрана. Баг, предложение или вопрос — попадает в админку.

**Тёмная и светлая тема.** Переключатель в настройках.

### Для организаторов

**Два типа событий.** Турнир (полная форма: название, формат, дата, место, мин/макс, взнос, описание) и быстрая встреча (формат, когда, где — за 10 секунд, исчезает через 24 часа).

**Шаблоны для повторяющихся событий.** Создаёшь шаблон "каждый четверг Pauper" с выбором дней недели и датой завершения. Клонируешь одной кнопкой — все поля заполнены, меняешь только дату.

**Отмена события.** Одна кнопка с подтверждением — все записавшиеся (going / maybe / waitlist) получают push об отмене.

**Сообщения участникам.** Написать всем записавшимся прямо из приложения: "начинаем позже", "нужен ещё 1 игрок", "локация изменилась". Push + внутренняя почта.

**Репутация организатора.** Статистика: сколько провёл ивентов, процент отмен, средняя посещаемость. Видна на профиле — игроки видят стоит ли идти.

**Рекомендации при создании.** После создания события платформа показывает список подходящих игроков. Чекбоксы, одна кнопка — приглашения отправлены.

### Для клубов

**Публичная страница клуба.** Логотип, описание, фотогалерея с просмотром в полном размере. Название, адрес, часы работы, вместимость, поддерживаемые форматы. Кнопка "Подписаться" — игроки получают уведомления обо всех событиях клуба.

**Контакты.** Структурированные контакты с кликабельными ссылками: сайт, телефон, группа WhatsApp, Facebook, Instagram. Всё в одном месте.

**Раздел "Как найти".** Интерактивная карта с меткой клуба. Кнопки "Открыть в Google Maps" и "Открыть в Waze". Текстовое описание проезда, ориентиры рядом. Отдельные фото входа и окрестностей — чтобы новый гость нашёл дверь.

**Карта клубов.** Все клубы на интерактивной карте с маркерами. Автоматически показывает ближайшие по геолокации. Клик по маркеру → переход на страницу клуба.

**Аналитика.** Уникальные игроки за месяц, популярные форматы по дням, пиковые дни, retention (кто вернулся повторно). Данные которых у клуба раньше не было.

**Все события в одном месте.** И турниры, и спонтанные встречи — видны на странице клуба. Понятно что происходит и когда.

### Админ-панель

Отдельная секция с навигацией. Дашборд с тремя зонами: алерты сегодня (неактивные организаторы, зависшие LFG-сигналы), метрики за неделю (новые пользователи, activation rate, события, RSVP, конверсия LFG, silent exits), тренды за 14 дней. Управление ролями пользователей (email + ссылка WhatsApp), редактирование тегов настроения, обработка баг-репортов.

### Языки

Английский, русский и иврит. Переключатель в настройках, автоопределение при первом входе.

## Что планируется

### Ближайшее

**Автоматическая генерация повторяющихся событий.** Шаблон уже можно создать — автоматическое создание экземпляров по расписанию в разработке.

**Подтверждение участия.** Push за 24 часа и за 3 часа до начала: "ты точно идёшь?". Одна кнопка. Организатор видит кто подтвердил а кто нет. Без штрафов.

**Добор игроков.** За 48 часов до события осталось 1-2 места — автоматический push подходящим игрокам.

**Готовые сообщения.** Шаблоны для организаторов: "Ивент подтверждён!", "Осталось 2 места", "Изменение локации" — отправка одним тапом.

**Карточки для шеринга.** Ссылка на событие в WhatsApp отображается с красивой картинкой (формат, дата, место, заполненность). После ивента — итоговая карточка "8 игроков, Pauper, Rotemz" для шеринга.

**Профиль стиля игры.** Мини-анкета: казуал/соревновательный, скорость, общительность. Помогает найти подходящих игроков.

### Среднесрочное

**Очки опыта (XP) и уровни.** Игроки получают XP за посещение событий, организацию, приглашение друзей. Уровни от Squire до Eternal. Бейджи на профиле. Мотивация через прогресс, не через наказания.

**Достижения.** "Первая игра", "5 ивентов за месяц", "Посетил 3 клуба", "Организатор" — автоматические бейджи на профиле.

**Реферальная система.** Пригласил друга — XP обоим. Встроенный вирусный рост.

**Календарь клуба.** Недельный/месячный вид: турниры, встречи, свободные слоты. Резервация времени под турнир — чтобы спонтанные встречи не налезали.

**Рейтинг клубов.** Публичный топ по активности и посещаемости. Дружеская конкуренция.

### Долгосрочное

**Лиги и сезоны.** Лига на 2-4 недели с таблицей результатов. Удерживает игроков — возвращаются чтобы закрыть сезон.

**Утилиты для игры.** Счётчик жизней (Commander с трекингом командирского урона), рандомизатор первого игрока. Игрок не уходит в другое приложение.

**WhatsApp/Telegram бот.** Команды из чата: /events today, /lfg pauper. Рост без убеждения ставить приложение.

**Carpooling.** "Еду из Тель-Авива, есть места" / "Ищу подвозку" на странице события.

**Подписка для клубов (Pro).** Расширенная аналитика, приоритетное отображение, брендированные ивенты, программа лояльности (баллы за посещения → призы).

**RTL-интерфейс.** Полная поддержка Right-to-Left для иврита.

**Нативное приложение.** Иконка в телефоне, push без ограничений.

## Что это даёт клубу

**Больше игроков на событиях.** Платформа находит подходящих людей и уведомляет автоматически. Организатору не нужно вручную обзванивать WhatsApp-контакты.

**Меньше no-show.** QR check-in, рейтинг надёжности, подтверждение за 24 часа — организатор знает кто реально придёт.

**Данные.** Аналитика: кто ходит, какие форматы популярны, в какие дни пик. Раньше это было в голове организатора, теперь — в цифрах.

**Привлечение новых.** Игроки находят клуб через карту, события, подписываются, приходят. Каждый шеринг в WhatsApp — бесплатная реклама с ссылкой на клуб.

**Профессиональная страница.** Логотип, галерея, контакты, карта, описание проезда — клуб выглядит серьёзно. Новый игрок сразу понимает что за место и как туда попасть.

**Ноль рутины.** Шаблоны событий, клонирование, готовые сообщения — организатор тратит минуту вместо получаса.
