# MTGx — Выжимка из исследований рынка

**Цель:** идеи для развития продукта после MVP. Акценты: не усложнять для игроков, никакого пэйвола/штрафов для игроков, монетизация только через магазины.

---

## Контекст: WhatsApp-дипломатия в Израиле

В израильской среде WhatsApp — доминирующий канал координации на всех уровнях, от государственных структур до локальных инициатив (Nefesh B'Nefesh и др.). MTG-сообщества полностью подчинены этой парадигме: клубные чаты, группы по форматам, спонтанные приглашения — всё через WhatsApp.

**Стратегический вывод:** попытка заменить WhatsApp внутренним чатом — фатальная ошибка. MTGx работает как утилитарный плагин для мессенджера: генерация deep link с превью (название, формат, прогресс-бар мест) → отправка в клубный чат → один тап для RSVP. Следующий шаг — WhatsApp/Telegram бот для кросс-постинга и управления без открытия приложения.

При проектировании уведомлений учитывать культурный контекст: гибкий режим тишины (Шаббат), мультиязычность (RU/EN/HE), горизонтальная координация через существующие группы а не вертикальная через новую платформу.

## Сильные конкуренты и референсы

### Прямые конкуренты

**MTG Finder** (mtg-finder.org)
Каталог игроков с фильтром по форматам, чат с игроками до встречи, уведомления о новых игроках в твоём формате. Ближайший конкурент по духу. Слабее по событийной логике, нет RSVP-системы, нет reliability, нет разделения на типы событий.

**Nerd Culture** (nerdculture.com/play-mtg)
Серьёзный конкурент с широким функционалом. Позиционируется как платформа для поиска игроков и групп по MTG и другим TCG. Что у них есть:
- Поиск игроков по формату, стилю игры (casual/competitive), уровню навыков
- Организация игровых сессий (онлайн и офлайн)
- Группы и чат как часть комьюнити-слоя
- Обмен колодами, трейдинг
- Матчинг по плейстайлу и скиллу — заявлен как ключевая ценность
- Охват не только MTG, но и другие TCG

**Что изучить у Nerd Culture:** их фильтры по стилю игры и матчинг по ожиданиям — это то, что MTGx пока не делает. Их слабость — размытый фокус (много игр), нет чёткого разделения турнир/спонтанная встреча, нет reliability score, нет интеграции с WhatsApp. MTGx побеждает за счёт специализации на оффлайн-координации и израильского рынка, но функционал матчинга по ожиданиям стоит перенять.

**TCG Compass** (tcgcompass.app)
Карта магазинов и ивентов, подписки на магазины и события, лента новостей/объявлений магазина, функция "claim your store" для владельцев. Конкурент по discovery-части. Не имеет LFG и спонтанных встреч.

### Сильные референсы (не прямые конкуренты, но ценные механики)

| Проект | Ценная механика для MTGx |
|---|---|
| **GameFor** | Публичные страницы магазинов с часами/событиями, группы для комьюнити, LFG-объявления. Лучший референс по клубному функционалу |
| **MeepleMatch** (meeplematch.app) | Авто-отмена при ненабранном минимуме к дедлайну, karma за неявки, min/max игроков. Референс по механикам надёжности |
| **Convoke** (convoke.games) | Commend/block/kick, позитивная игра. Для MTGx офлайн-версия: репутация и быстрый бан у конкретного организатора |
| **SpellBot** (spellbot.io) | Discord-бот с /lfg. Референс по дистрибуции LFG через ботов без отдельного приложения |
| **TopDeck.gg** | Лиговый режим с LFG, публичные лобби, история матчей. Референс по лигам/сезонам как промежуточному формату между турниром и встречей |
| **Aftergame** | Календарь, статистика, нотификации по играм из вишлиста. Паттерн: подписался на формат → push когда кто-то собирает это играть |
| **GameTree** | Матчмейкинг по совместимости через мини-тест. «Тиндер для настолок» — подсветка игроков с похожим стилем |

**Wizards Companion / EventLink** — соседняя ниша, не конкурент. Операционка для санкционированных турниров, не discovery. Нет LFG. Высокий порог входа (аккаунт WotC).

**Melee.gg** — тяжёлая турнирная система. Валидация деклистов, тай-брейкеры. Оверхед для казуальных встреч.

## Идеи для развития MTGx

### Высокий приоритет (усиливает core — LFG, удобство, явку)

**Intent Tags на быстрых встречах.** При создании quick meetup — выбор тега: «Тестирование колоды», «Казуальная игра», «Подготовка к турниру». Снижает неопределённость, повышает конверсию. Игрок понимает куда идёт.

**Двухфазная запись (мягкая).** За 24–6 часов до начала приходит push «Подтверди участие» с одной кнопкой. Не штраф — просто уточнение. Кто не ответил — остаётся в maybe, организатор видит реальную картину.

**Авто-отмена при ненабранном минимуме.** За 3 часа до старта если min_players не набран — событие отменяется, все получают уведомление. Никто не приходит зря.

**Waitlist.** Когда max_players достигнут — игрок встаёт в очередь. Если кто-то отменяет — первый из waitlist получает push и автоматически переходит в going.

**Proxy Policy на событиях.** Переключатель: "No Proxies" / "X Cards Limit" / "100% Proxy Friendly". Устраняет типовые вопросы.

**Язык стола.** Флаг на событии: RU / HE / EN. Критично для Израиля.

**Reliability score — видимость без наказания.** Текущая формула работает. Добавить: отображение статуса («новичок» / «надёжный» / «рискованный»). Видно только организаторам. Никаких блокировок или ограничений для игроков — только информация для организатора.

### Средний приоритет (клубы, организаторы, удержание)

**Регулярные события.** Серия с шаблоном и паттерном (каждый вторник), возможность паузы. Must-have для клубов.

**Базовая аналитика для клуба.** Уникальные игроки в месяц, популярные форматы по дням, retention. Ценность для владельца — видеть эффект от платформы.

**Коммуникация организатор → участники.** Push всем записавшимся: «начинаем позже», «освободилось место». Быстрые шаблоны.

**Интеграция с календарями.** «Добавить в Google Calendar / iCal» на странице события.

**Геймификация: достижения и очки опыта.** Заложить систему XP и ачивок с самого начала, даже если на старте будет минимальный набор:
- XP за RSVP going + реальную явку
- XP за создание события которое состоялось
- XP за приглашение нового игрока (виральность)
- Ачивки: «Первая игра», «5 игр в месяц», «Организатор» (создал 3 события), «Мультиформат» (играл в 3+ форматов), «Надёжный» (10 подряд без отмен)
- Уровни профиля (визуальный бейдж)
- Не привязывать к деньгам или привилегиям — чисто статусная мотивация и fun

Создаёт привычку открывать приложение и даёт ощущение прогресса. Данные XP — фундамент для будущих программ лояльности клубов.

### Низкий приоритет (после product-market fit)

**Commander Brackets.** 5 уровней силы колод. При RSVP на Commander — опционально ссылка на деклист (Moxfield/Archidekt), автооценка bracket через API.

**Плейстайл-профиль.** Анкета: casual/competitive, скорость, общительность. Подсветка совместимости. Референс: Nerd Culture, GameTree.

**Сессионные утилиты.** Счётчик жизней (Commander с командирским уроном), рандомизатор первого игрока. Удерживает в приложении во время игры.

**Лиги/сезоны.** Лига на 2–4 недели с гибкими матчами. Промежуточный формат. Монетизируется через клубы.

**Программа лояльности для клубов (Prize Wall).** Клуб начисляет баллы (привязка к XP-системе), игроки обменивают на призы. QR-код для чек-ина. Оплачивается клубом.

**WhatsApp/Telegram бот.** Команды: /events today, /lfg pauper tomorrow. Подтверждение/отмена из бота.

**Геолокационный поиск.** Радиус +15 км вместо жёсткой фильтрации по городу. Для Гуш-Дана объединяет смежные города.

## Принципы развития

**Для игроков — бесплатно, просто, интересно.** Никаких штрафов, ограничений, пэйволов. Мотивация через геймификацию (XP, ачивки, бейджи), не через наказания. Reliability score — информация для организатора, не кнут для игрока.

**Монетизация — только через магазины.** Игроки не платежеспособны. Ценность для клубов: аналитика, автоматизация, привлечение аудитории, программы лояльности. Freemium B2B: базовый функционал бесплатно, Pro-фичи за подписку клуба.

**Симбиоз с WhatsApp, не конкуренция.** Платформа — утилитарный плагин для существующих каналов коммуникации. Учитывать израильскую специфику: WhatsApp-дипломатия, Шаббат, мультиязычность.

**Фокус — координация встреч.** Не соцсеть, не трейдинг, не новостная лента. Тонкий инструмент: создал → позвал → пришёл → поиграл.

## Что НЕ делать

- Общая лента постов, мемы, новости — есть Discord и WhatsApp
- Трейдинг/обмен картами — другой продукт
- Универсальный чат — убивает фокус
- Субъективные оценки игроков (звёзды/отзывы) — сведение счётов
- Штрафы и блокировки для игроков — отпугивает аудиторию
- Платный функционал для игроков — убивает adoption
- Депозиты/предоплата с игроков — если нужно, клуб решает сам вне платформы
